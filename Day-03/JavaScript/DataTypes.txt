Data types in JS

there are two types of Datatypes in JS

1. Primitive Data Types :- Number,String,Undefined,Boolean,Null etc are Primitive Data types

Number :- let num = 30;

String :- let str = "Hello World"

Undefined :- let uninitialised

Boolean :- let isPrime = true;

Null :- let emptyValue = NULL;


this are known as primitive data types

2. Non- Primitive Data Types :- Array,Object,function,date etc are non-primitive Datatypes

Array :- Represent the similar types of datatypes element

Eg. let arr1 = [1,2,4,5,5,5,3,3,2,3,4];

Object :- Represent the mix types of datatypes element

Eg. let obj1 = {
    name : "Yash Jain",
    age : "23",
    DOB : "30-10-2000",
    id : 1
}


Function :- Represent a block of code to perform a particular task

Eg. function greet(){
    console.log("Hello");
}

Date :- Represent data and time

Eg. let now = new Date();
